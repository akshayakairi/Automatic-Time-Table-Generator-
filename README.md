# Automatic-Time-Table-Generator-
Developed a web-based Automatic Timetable Generator that creates class schedules based on user-defined subjects, faculty assignments, number of periods per day, and weekly subject requirements. The system automatically allocates subjects to available periods while incorporating lunch breaks and generating a structured, printable timetable

## Features

* Distributes subjects across the week
* Prevents subject collisions
* Includes lunch breaks
* Allocates free periods automatically
* Generates a balanced timetable

## Technologies Used

 * Python
 * HTML
 * CSS
 * JavaScript

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run:

```bash
python main.py
```

## Sample Input

* Subjects: Math, Physics, Chemistry, English
* User inputs: Number of working days per week, school start time, school end time, lunch break duration, and subject details

## Output

A complete weekly timetable with subjects, free periods, and lunch breaks.
